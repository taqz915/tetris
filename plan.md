# 테트리스 웹 애플리케이션 MVP 계획

## Context
사용자가 브라우저에서 바로 실행 가능한 간단한 테트리스 게임을 요청했습니다. 고급 기능 없이 기본적인 게임 플레이(블록 이동, 점수 표시)만 제공하는 MVP 버전을 만듭니다. 현재 디렉토리는 비어있으며, 처음부터 새로 구축합니다.

## Implementation Plan

### 파일 구조
```
tetris/
├── index.html (메인 HTML 파일)
├── style.css (스타일시트)
└── game.js (게임 로직)
```

### 1. index.html 생성 (구조만)
- 기본 HTML5 구조
- `<link>` 태그로 style.css 연결
- `<script>` 태그로 game.js 연결 (defer 속성)
- 게임 캔버스 (`<canvas>` 요소)
- 점수 표시 영역 (`<div>` 요소)
- 간단한 게임 설명 (키보드 컨트롤 안내)
- **스타일 코드 없음** (모든 스타일은 style.css에)
- **JavaScript 코드 없음** (모든 로직은 game.js에)

### 2. style.css 생성 (스타일만)
- body, html 기본 스타일
- 중앙 정렬된 게임 컨테이너 레이아웃
- 캔버스 스타일링 (테두리, 배경색, 그림자)
- 점수 표시 영역 스타일 (폰트, 크기, 색상)
- 게임 설명 텍스트 스타일
- 반응형 디자인 (모바일에서도 볼 수 있도록)
- 깔끔하고 미니멀한 디자인
- **HTML에 인라인 스타일 없음**

### 3. game.js 생성 (게임 로직만)
- **HTML에 JavaScript 코드 없음** (모든 로직은 이 파일에)
- DOM 요소 참조 (canvas, 점수 표시 요소)
- 모든 게임 로직과 이벤트 핸들러

#### 핵심 기능:
1. **게임 보드 설정**
   - 10x20 그리드 (표준 테트리스 크기)
   - 2D 배열로 보드 상태 관리

2. **테트로미노 (블록) 정의**
   - 7가지 기본 테트로미노 형태 (I, O, T, S, Z, J, L)
   - 각 블록의 회전 상태 배열
   - 색상 지정

3. **블록 이동 및 제어**
   - 키보드 이벤트 리스너:
     - 왼쪽/오른쪽 화살표: 좌우 이동
     - 아래 화살표: 빠른 낙하
     - 위 화살표/스페이스바: 회전
   - 충돌 감지 (벽, 바닥, 다른 블록)
   - 자동 낙하 (setInterval 사용)

4. **줄 제거 로직**
   - 완성된 줄 감지
   - 줄 제거 및 위 블록들 아래로 이동
   - 점수 증가 (한 줄당 100점)

5. **게임 상태 관리**
   - 점수 추적 및 업데이트
   - 게임 오버 감지 (새 블록이 생성 위치에서 충돌)
   - 게임 오버 시 알림 및 재시작 옵션

6. **렌더링**
   - Canvas API로 그리드, 블록, 현재 테트로미노 그리기
   - requestAnimationFrame 또는 setInterval로 게임 루프

### MVP 범위 (포함하지 않음)
- 다음 블록 미리보기
- 레벨 시스템
- 하드 드롭 기능
- Hold 기능
- 효과음/배경음악
- 모바일 터치 컨트롤
- 최고 점수 저장

## 검증 방법
1. 브라우저에서 `index.html` 파일 열기
2. 게임이 자동으로 시작되는지 확인
3. 키보드 컨트롤 테스트:
   - 좌우 화살표로 블록 이동
   - 위 화살표로 블록 회전
   - 아래 화살표로 빠른 낙하
4. 줄 완성 시 제거 및 점수 증가 확인
5. 게임 오버 조건 테스트 (블록이 맨 위까지 쌓임)
6. 다양한 브라우저에서 호환성 확인 (Chrome, Firefox, Safari)

## 기술 스택
- 순수 HTML5, CSS3, JavaScript (프레임워크 없음)
- Canvas API (게임 렌더링)
- 외부 라이브러리 없음 (브라우저에서 바로 실행 가능)
